# pyfolio
TUI-based portfolio optimization tool that uses Markwotiz's Portfolio Theory to find optimal weights for a combination of assets that maximizes Sharpe. 5-year lookback data with ``yfinance``.

<img width="928" height="596" alt="image" src="https://github.com/user-attachments/assets/c157ed61-015b-45c0-b006-0157ebe086ba" />



## Usage
Clone the repository:
```
git clone https://github.com/mariohaya/pyfolio.git
```
When prompted, enter one or more ticker symbols separated by spaces:
```
Ticker: AAPL AMD NVDA AMZN
```
Enter risk-free rate:
```
0.04
```
